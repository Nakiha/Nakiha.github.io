CSS摘记-选择器

Posted on 2019-08-14 | Edited on 2019-08-15

原文来自 MDN的CSS学习指南

CSS 选择器

  • @-规则
    @media @supports @document

  • 类型/元素选择器
    p {}

  • 类选择器
    .className {}
    (className是html中lclass属性的值)

  • ID选择器
    #id {}
    (id是html中id属性的值)

  • 存在和值属性选择器
    [attr]：该选择器选择包含 attr 属性的所有元素，不论 attr 的值为何。
    [attr=val]：该选择器仅选择 attr 属性被赋值为 val 的所有元素。
    [attr~=val]：该选择器仅选择具有 attr 属性的元素，而且要求 val 值是 attr 值包含的被空格分隔的取值列表里中的一个。

  • 子串值 属性选择器
    [attr|=val] : 选择attr属性的值是 val 或值以 val- 开头的元素
    [attr^=val] : 选择attr属性的值以 val 开头（包括 val）的元素。
    [attr$=val] : 选择attr属性的值以 val 结尾（包括 val）的元素。
    [attr*=val] : 选择attr属性的值中包含子字符串 val 的元素

  • 伪类 (添加到选择器末尾的关键字)
    a:hover, a:active, a:foucs {}
    a: visted{}
    li:nth-of-type(2n+1){} nth-child()

  • 伪元素 (与伪类很像, 用于选择某个元素的某个部分)
    [href^= http ] :: after {

    content: '⤴'

    }

  • 组合器和选择器组

    • 匹配 A/B
      A. B (table td, table th 不是组合器, 只是允许将几个选择器使用相同的CSS规则)
    • 匹配 B
      A B (B为A的后代节点)
      A>B (B为A的直接子节点)
      A+B (B是A的下一个同级节点)
      A~B (B是A之后的同级节点)

CSS box 模型

  • box-sizing 属性

    • 默认时的 content-box

    • 可选的 border-box

  • box 部分操作

    • overflow 属性
      当使用绝对值设置了box大小, 而内容过多时, 默认情况下内容会超出box
      visible (默认)
      hidden (裁剪)
      auto (出现滚动条)

    • background-clip 属性
      背景通常是以box的border为边界进行裁剪, 需要变更时可以更改
      border-box (默认)
      padding-box
      content-box

    • display 属性
      在box中的元素也会有自己的box, 我们可以为它设置相关属性
      inline 随着文档的文字流动, 不破坏原有行间距
      block 另起一行
      inline-block 会换行和改变j行间距的inline

CSS 层叠关系

重要性 > 专用性 > 源代码次序

  • 重要性
    为属性后面添加 !important

  • 专用性
    加分制: 选择器的专用性由四个值决定, 可以b将其认为是4个整数 0 0 0 0 起始

    1. 直接将样式写在html元素的style属性中 + 1 0 0 0 分
    2. ID选择器 + 0 1 0 0 分
    3. 类选择器, 属性选择器, 伪类 + 0 0 1 0 分
    4. 元素选择器或者伪元素 + 0 0 0 1 分

  • 源代码次序
    如果重要性和专用性相同, 则写在后面的表生效
