Dart 与 Java 在写法上的区别

Dart不是像Java那样必须是由一个一个类构成的语言, 结构写法上更像 JS 和 C++, Dart的缩进也不是四个空格或者/t, 而是两个空格, 在安装Dart SDK 后 dartfmt 可以帮助我们完成这种特殊的缩进.

特性

  • 没有重载 !

    Dart 和 C# 一样提供了可选参数, 用于放弃重载, 其语法基本和 C#(疯狂语法糖的屑语言) 的指定参数基本相同

  • 所有类未初始化时都是null !

  • 每个类都定义了一个接口 !

    Dart 语言并没有提供 interface 关键字，但是每一个类都隐式地定义了一个接口。所以没有extends只有implement.

  • Dart没有数组只有泛型集合 !

  • 可以在集合里塞判断和循环

  • Dart 如同 C++ 一样, 提供了函数指针

  • 提供了var关键字, 来容纳可以被推导出的类型 ( JS再世? )

  • final和const

    const表示这个变量编译时可确定

  • Dart 2 之后实例化对象的new变成了可选的.

  • 简化 return 语句

  • 简化私有声明

    所有名字以下划线开头的变量，Dart 的编译器都会将其强制标记为私有的

  • 简化构造器

    Dart中, 如果构造器可以没有构造体, 如果不需要对传入的值进行处理的话.

  • 简化工厂方法

    Dart 提供了factory关键字专门用来修饰工厂方法

  • 提供了像C#一样的 async/await 关键字, 来简化一些异步代码

  • String是 UTF-16 , 但是提供了Runes类来表示 UTF-32 (Emoji或成最大赢家 \u{1f60e} )

  • Dart 的匿名甚至可以让你在foreach时不用声明子项的类型
