Python特性速浏

Posted on 2019-09-16 | Edited on 2019-09-24

花几天的时间了解下Python的语法和结构, 以备不时之需

列表

  • 初始化列表使用[value, ...]

    ['1','2','3']

  • 列表默认就可以视作栈或者链表

    • Tips

      列表为空时使用if得到false, 否则true

    • 更改元素

      .insert(),.append(), .pop(), del array[x], remove(x)

    • 列表整体操作

      .sort() .sorted()

    • 查询

      x in array, len()

字典

  • 初始化字典使用{key:value, ...}

    1. 如果一开始就可以确定字典中的项, 那么初始化带有全部数据的字典
    2. 如果需要动态添加项, 那么初始化空字典dictionary = {}

  • 字典为键值对结构, 直接对字典中不存在的键赋值就可以添加项, 删除项仍旧使用del dictionary[key]的模式

    • 直接使用字典名作为for in的循环参数时, 将遍历所有的键, 如果要指定获得键, 使用字典的keys()函数

    • 字典的常用函数

      keys(), values()

函数

  • 定义函数

  • 返回值

    一个函数允许返回任何类型的值, 只要它使用了return

  • 传递不定数目的实参

  • 传入不定数目键值对, 自动生成字典

创建与导入模块

  • 写好的函数保存在 .py 文件中, 如果需要复用, 使用import导入, 这种方法需要使用模块名.函数名的方式使用函数
  • 如果只需要导入某一个函数, 使用 from ... import ...
  • 在import或者from import语句函数后面添加as关键字可以添加别名
  • 比较少用的一种方式是直接导入全部的函数和类, form module_name import *

创建与使用类

  • class关键字用来标识类

  • __init__()函数是所有类的初始化函数

  • 如果使用继承, 给类命中的括号中, 添加集成的父类

    class Son(Father):

文件相关

  • 为opne() close()这两个函数传入相对路径或者绝对路径可以返回一个文件, 利用as可以轻松为其增加别名

  • Python 提供了with关键字用来保证文件的关闭

  • 可以为open()提供第二个值作为附加参数

    • 'r' (默认的) 只读模式
    • 'w' 写入模式 (如果要写入的文件不存在, 那么抛出异常)
    • 'a' 附加模式
    • 'r+' 读写模式

异常处理

  • try-except-else代码块被用来代替try-catch-fanally

  • 如果不想进行任何异常处理, 可使用pass关键字, 直接跳过异常处理

储存 JSON 数据

  • 通过引入json包我们可以使用json包提供的函数快速保存与从json文件中取数据

测试

  • python 的单元测试库为 unittest, 下面给出一个用例

  • 各种各样的断言方法

    • assertEqual(a, b) a == b
    • assertNotEqual(a, b) a!=b
    • assertTrue(x) x==true
    • assertFalse(x) x==false
    • assertIn(item, list) item in list
    • assertNotIn(item, list) !(item in list)

杂物

  • 快速获取无重复元素集合 set()

  • foreach语句

    for item in array:

  • range()快速生成顺序数字函数

    • Tips

      1. 使用list()将range()生成的值取为列表

      2. 使用for in语句执行列表解析

      3. range()接在普通for in语句后面可以用来控制循环次数

  • 列表切片

    对于数组, 方括号中使用 start:end可获得 sub Array

    • Tips

      使用空切片可以得到一个列表的浅复制

  • and,or分别代替&&和||

  • if语句没有括号与, 并且if else被替换为if-elif-else结构
