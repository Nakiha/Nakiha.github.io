Class类

Class没有公共构造函数, 它在类加载器加载一个类进虚拟机时被自动构建,

处理 .class 的逻辑

  1. 装载:

    类加载器将.class的二进制文件装进JVM的方法区, 并在堆区创建描述这个类的java.lang.Class对象, 每个类只会被装载一次.

  2. 链接

    用于将二进制文件组装为可运行的状态, 分为

    1. 校验

      用于确认此二进制文件是否适合当前 JVM.

    2. 准备

      为静态成员分配内存空间, 并设置默认值.

    3. 解析

      转换常量池中的代码为直接引用.

  3. 初始化

    对类中的变量进行初始化值, 使对象可以被正常使用, CG时, 如果对象不被引用, 它将被卸载.再次使用时重新被装载, 链接和初始化.

这些步骤说明, 类只会被装载一次, 通过.class对象虚拟机就可以实例化出各种不同的对象.
